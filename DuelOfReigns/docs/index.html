RequireAuth

A component that wraps its children and ensures that the user is authenticated. If the user is not authenticated, redirects to the login page and saves the current location.

RequireAuth
Parameters
props (Object) The props object.
Name Description
props.children React.ReactNode The components to render if the user is authenticated.
Returns
React.ReactNode: Returns the children if authenticated, otherwise navigates to the login page.

SkinBuy

SkinBuy component renders a card for purchasing a skin item. It displays skin details, including the seller, price, rarity, and provides functionality for authenticated users to buy the skin. The component includes loading states, tooltips, and alerts for interaction feedback.

SkinBuy
Parameters
item (Object) The skin item to be displayed and purchased.
Name Description
item.skin Object The details of the skin, including image and name.
item.rarity number The rarity level of the skin (0: None, 1: Common, 2: Rare, 3: Epic, 4: Legendary).
item.user_seller Object The seller of the skin.
item.user_seller.name string The name of the skin's seller.
item.skin_sale Object The sale details of the skin.
item.skin_sale.price_without_commission number The base price of the skin.
item.skin_sale.fee number The fee percentage applied to the base price.
item.skin_sale.id string The unique identifier for the skin sale.
item.item any
Returns
JSX.Element: The rendered skin buy card component.

SkinSell

MarketPage

MarketPage component displays a marketplace where users can view and purchase skins available for sale. The component fetches skin sales data from a Firestore database, including details about each skin and the seller. It also provides an option for authenticated users to navigate to a selling page to list their own items for sale.
